Vue.js

Vue (pronuncia-se /vju/, como view, em inglês) é um framework progressivo para a construção de interfaces de usuário.

Ao contrário de outros frameworks monolíticos, Vue foi projetado desde sua concepção para ser adotável incrementalmente.

biblioteca principal é focada exclusivamente na camada visual (view layer), sendo fácil adotar e integrar com outras bibliotecas ou projetos existentes.

Por outro lado, Vue também é perfeitamente capaz de dar poder a sofisticadas Single-Page Applications quando usado em conjunto com ferramentas modernas e bibliotecas de apoio.
Ele se baseia em cima do HTML padrão, CSS e JavaScript, e fornece um modelo de programação declarativo e baseado em componentes que ajuda você a desenvolver eficientemente interfaces de usuário

Relação com elementos customizados

Componentes Vue são muito similares aos Elementos Customizados, parte da Especificação de Web Components.
Isto ocorre pois a sintaxe de componentes Vue foi vagamente modelada a partir da especificação.

A Especificação de Web Components foi finalizada, mas não está nativamente implementada em todos os navegadores.
O Safari 10.1+, Chrome 54+ e Firefox 63+ suportam nativamente os componentes da web. Em comparação, componentes Vue não requerem qualquer tipo de polyfill e funcionam consistentemente em todos os navegadores suportados (IE9 e superiores).
Quando necessário, componentes Vue também podem ser envolvidos dentro de um elemento customizado nativo.

Componentes Vue oferecem importantes recursos não disponíveis em elementos customizados tradicionais, mais notavelmente: fluxo de dados entre componentes, comunicação com eventos customizados e integração com ferramentas para build.

Embora o Vue não use elementos personalizados internamente, ele tem ótima interoperabilidade quando se trata de consumir ou distribuir como elementos personalizados.
